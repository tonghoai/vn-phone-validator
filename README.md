# VN-PHONE-VALIDATOR
vn-phone-validator là một package hỗ trợ xử lý số điện thoại đầu số ở Việt Nam. Nó đơn giản, mạnh mẽ & dễ sử dụng.

### v2.x release
- Cải tiến cú pháp, tốc độ

## Installation
```sh
$ npm i vn-phone-validator
```
